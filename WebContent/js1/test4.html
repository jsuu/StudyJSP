연산자

1. 산술연산자

+ - * / %

2. 대입연산자

변수에 데이터를 정하는 연산 =(대입연산자), +=, -=, *=, /=, %= (복합대입연산)

1 2 3

3. 증감연산자 (단항연산자)

숫자데이터를 1증가, 감소시키는 연산자 ++A / A++, --A / A--

4. 비교연산자 (단항연산자)

두개의 데이터 비교. 해당연산의 결과는 boolean-true/false > < >= <= == != ===(값과 데이터도 비교 모두같아야 true) !== (값과 데이터도 비교 모두달라야 true) >= 와 => 는 다르다. => =이 대입연산이므로 오른쪽에 연산이 있어서는 안됨 는 >비교연산을 왼쪽에 대입한다는 의미

5. 논리연산자

논리값(t/f)을 이용 연산을 처리하는 연산자 &&(and) ||(or) !(not)

6. 삼항 조건연산자

조건식의 결과에 따라 실행하는 코드가 변경되는 연산자 조건식? 실행문1(참) : 실행문2(거짓);
